# STS3215 Motor Test & ID Setup - 사용자 매뉴얼

## 프로그램 소개

STS3215 Motor Test는 Feetech STS3215 서보 모터를 제어하고 테스트하기 위한 애플리케이션입니다. 직관적인 인터페이스를 통해 모터 연결, 제어, 실시간 모니터링, 모터 ID 설정을 쉽게 수행할 수 있습니다.

프로그램 시작 시 **테스트 모드 선택 화면**이 나타나며, 단일 모터 테스트 또는 SO-ARM 101 전체 모터 테스트 중 하나를 선택할 수 있습니다.

---

## 시작 화면 — 모드 선택

프로그램을 실행하면 먼저 테스트 모드를 선택하는 다이얼로그가 나타납니다.

| 버튼 | 설명 |
|------|------|
| **단일 모터 테스트** | 하나의 모터를 선택해 세부 제어 및 모니터링 |
| **SO-ARM 101 전체 모터 테스트** | SO-ARM 101 로봇암의 6개 모터를 동시에 모니터링 및 제어 |
| **종료** | 프로그램을 종료합니다 |

---

## 모드 1 — 단일 모터 테스트

### 화면 구성

```
┌─────────────────────┐
                      STS3215 Motor Test                    
├─────────────────────┤
      [연결]              포트 선택 및 연결/해제         
├─────────────────────┤
      [모터 선택]      모터 스캔 및 선택              
├─────────────────────┤
      [모터 ID 설정]  모터 ID 변경                   
├─────────────────────┤
      [모터 제어]      위치, 속도, 가속도, 토크 제어   
├─────────────────────┤
      [상태 모니터]   실시간 모터 상태 확인           
├─────────────────────┤
      [로그]              작업 기록 및 오류 메시지             
└─────────────────────┘
```

### 기능별 사용법

#### 1. 연결

모터와 통신하기 위해 먼저 시리얼 포트에 연결해야 합니다.

**사용 방법:**
1. USB 케이블을 컴퓨터에 연결합니다
2. **포트 선택** 드롭다운에서 연결할 포트를 선택합니다
   - USB 포트가 자동으로 상단에 표시됩니다
3. **🔄 새로고침** 버튼으로 포트 목록을 갱신할 수 있습니다
4. **🔌 연결** 버튼을 클릭합니다
5. 연결 성공 시 LED 표시기가 초록색으로 변합니다

**연결 해제:**
- 연결된 상태에서 **연결 해제** 버튼을 클릭합니다

---

#### 2. 모터 선택

연결된 모터를 검색하고 제어할 모터를 선택합니다.

**사용 방법:**
1. **🔍 스캔** 버튼을 클릭합니다
2. 프로그램이 ID 1~29 범위에서 모터를 자동으로 검색합니다
3. 진행률 창이 표시되며, 완료 시 발견된 모터 수가 표시됩니다
4. **모터** 드롭다운에서 제어할 모터를 선택합니다
5. **📡 핑 테스트** 버튼으로 선택한 모터와의 통신을 확인할 수 있습니다

---

#### 3. 모터 ID 설정

모터의 ID를 변경할 수 있습니다.

**사용 방법:**
1. 변경할 모터를 선택합니다 (현재 ID가 표시됩니다)
2. **새 ID** 입력란에 변경할 ID를 입력합니다 (0~253)
3. **✏️ ID 변경** 버튼을 클릭합니다
4. 확인 창에서 **Yes**를 선택합니다
5. 변경 완료 후 **스캔**을 다시 실행하여 목록을 갱신합니다

> ⚠️ **주의:** ID 변경 후에는 반드시 모터를 다시 스캔해야 합니다.

---

#### 4. 모터 제어

모터의 위치, 속도, 가속도를 조절하고 이동 명령을 내립니다.

##### 파라미터 설정

| 파라미터 | 범위 | 설명 |
|----------|------|------|
| 위치 | 0 ~ 4095 | 모터의 목표 위치 |
| 속도 | 0 ~ 3400 | 이동 속도 (값이 클수록 빠름) |
| 가속도 | 0 ~ 254 | 가속도 레벨 (값이 클수록 급가속) |

**사용 방법:**
1. 슬라이더를 드래그하거나 숫자를 직접 입력하여 값을 설정합니다
2. **▶ 이동** 버튼을 클릭하면 설정된 값으로 모터가 이동합니다
3. **⏹ 정지** 버튼으로 이동 중인 모터를 즉시 정지시킵니다

##### 토크 제어
- **⚡ 토크 ON**: 모터에 토크를 활성화합니다 (모터가 힘을 유지)
- **토크 OFF**: 토크를 비활성화합니다 (모터가 자유롭게 회전)

---

#### 5. 상태 모니터

모터의 현재 상태를 실시간으로 확인합니다.

##### 표시 항목

| 항목 | 단위 | 설명 |
|------|------|------|
| 위치 | 0~4095 | 현재 모터 위치 |
| 속도 | - | 현재 회전 속도 |
| 온도 | °C | 모터 내부 온도 |
| 전압 | V | 공급 전압 |
| 전류 | mA | 소비 전류 |
| 부하 | % | 모터 부하율 |

**사용 방법:**
- **📊 모니터링 시작**: 자동으로 상태를 계속 갱신합니다
- **📖 1회 읽기**: 현재 상태를 한 번만 읽습니다

모니터링 중에 다시 버튼을 클릭하면 모니터링이 중지됩니다.

---

#### 6. 로그

모든 작업 내역과 오류 메시지가 기록됩니다.

- 연결/해제 상태
- 모터 스캔 결과
- 제어 명령 실행 결과
- 오류 발생 시 상세 메시지

---

### 기본 사용 흐름

```
1. 하드웨어 연결
   └ USB-시리얼 어댑터 + 모터 전원 연결

2. 프로그램에서 포트 연결
   └ 포트 선택 → 연결 버튼

3. 모터 스캔
   └ 스캔 버튼 → 모터 선택

4. 모터 제어
   └ 위치/속도/가속도 설정 → 이동

5. 상태 확인
   └ 모니터링 시작으로 실시간 확인
```

## 모드 2 — SO-ARM 101 전체 모터 테스트

SO-ARM 101 로봇암에 장착된 여러 STS3215 모터를 동시에 모니터링하고 제어하는 모드입니다.

### 화면 구성

```
┌─────────────────────────────────────────────┐
          SO-ARM 101 전체 모터 테스트              
├─────────────────────────────────────────────┤
      [연결 설정]   포트 선택, ID 목록 입력, 연결    
├─────────────────────────────────────────────┤
      [충돌 감지 설정]   임계값 설정 및 활성화       
├─────────────────────────────────────────────┤
      [모니터링 및 제어]                           
         모터 ID 1 카드 — 위치/부하/전류 + 슬라이더  
         모터 ID 2 카드                            
         ...                                      
└─────────────────────────────────────────────┘
```

### 기능별 사용법

#### 1. 연결 설정

**사용 방법:**
1. **포트** 드롭다운에서 연결할 포트를 선택합니다
2. **🔄 새로고침** 버튼으로 포트 목록을 갱신합니다
3. **ID 목록** 입력란에 제어할 모터 ID를 쉼표로 구분하여 입력합니다
   - 기본값: `1,2,3,4,5,6` (SO-ARM 101 기준)
4. **🔌 연결** 버튼을 클릭합니다
5. 연결 성공 시 LED 표시기가 초록색으로 변하고, 각 모터 카드가 표시됩니다
6. 다시 클릭하면 **연결 해제**됩니다

---

#### 2. 충돌 감지 설정

모터에 과부하 또는 과전류가 발생하면 자동으로 토크를 차단하여 모터와 로봇을 보호합니다.

| 항목 | 기본값 | 설명 |
|------|--------|------|
| 충돌 감지 활성화 | ON | 체크 해제 시 비활성화 |
| 부하 임계값 | 1300 / 1300 | 이 값 초과 시 토크 OFF |
| 전류 임계값 | 800 mA | 이 값 초과 시 토크 OFF |

**임계값 초과 시 동작:**
- 해당 모터 토크 자동 차단
- 모터 카드에 **⚠️ 충돌 감지!** 경고 표시
- 카드 테두리가 빨간색으로 변경
- 슬라이더를 움직이면 충돌 상태가 해제되고 토크가 다시 활성화됩니다

---

#### 3. 모터 카드 — 모니터링 및 제어

연결된 각 모터마다 개별 카드가 표시됩니다.

##### 표시 항목

| 항목 | 설명 |
|------|------|
| **현재 값** | 모터의 현재 위치 (0~4095) |
| **부하** | 현재 부하값. 70% 초과 시 주황색, 임계값 초과 시 빨간색 |
| **전류** | 현재 전류(mA). 70% 초과 시 주황색, 임계값 초과 시 빨간색 |
| **슬라이더** | 목표 위치를 설정하여 모터를 이동시킵니다 (0~4095) |
| **목표** | 슬라이더에 설정된 목표 위치 |

**실시간 갱신**: 100ms 주기로 자동 업데이트됩니다.

---

### 기본 사용 흐름

```
1. 하드웨어 연결
   └ USB-시리얼 어댑터 + SO-ARM 101 전원 연결

2. 연결 설정
   └ 포트 선택 → ID 목록 확인 (기본: 1,2,3,4,5,6) → 연결 버튼

3. 충돌 감지 설정
   └ 임계값 조정 (선택 사항)

4. 모터 제어
   └ 각 모터 카드의 슬라이더를 조작하여 목표 위치 설정

5. 실시간 모니터링
   └ 부하/전류 수치를 확인하며 안전하게 운용
```

## 문제 해결

### 포트가 목록에 나타나지 않음
- USB-시리얼 어댑터가 제대로 연결되었는지 확인하세요
- **🔄 새로고침** 버튼을 클릭하세요
- 어댑터를 다른 USB 포트에 연결해 보세요
- Linux의 경우: 로그아웃 후 다시 로그인하거나 컴퓨터를 재시작하세요

### 모터가 스캔되지 않음 (단일 모드)
- 모터에 전원이 공급되고 있는지 확인하세요
- 시리얼 케이블 연결 상태를 확인하세요
- 모터 ID가 1~29 범위 내에 있는지 확인하세요

### 모터가 움직이지 않음
- **토크 ON** 상태인지 확인하세요
- 속도 값이 0이 아닌지 확인하세요 (단일 모드)
- 모터 전원 공급 상태를 확인하세요
- SO-ARM 101 모드: 충돌 감지로 토크가 꺼졌을 수 있습니다 — 슬라이더를 움직여 재활성화하세요

### SO-ARM 101 연결 후 카드가 표시되지 않음
- ID 목록이 올바르게 입력되었는지 확인하세요 (예: `1,2,3,4,5,6`)
- 모터 전원과 시리얼 케이블 연결 상태를 확인하세요

### 연결이 불안정함
- USB 케이블을 교체해 보세요
- 다른 USB 포트에 연결해 보세요
- 시리얼 통신 케이블 접촉 불량을 확인하세요

### 프로그램이 실행되지 않음 (Linux)
- AppImage 파일에 실행 권한이 있는지 확인하세요
- 터미널에서 직접 실행하여 오류 메시지를 확인하세요
